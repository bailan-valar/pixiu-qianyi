# 貔貅记账转换工具 - 使用说明

## 功能介绍

这是一个图形界面工具，用于将您的记账数据转换为貔貅记账导入格式。

### 主要功能

1. **文件选择** - 导入 Result_2.csv 格式的源文件
2. **分类映射** - 配置原分类到貔貅记账分类的映射关系
   - 实时搜索功能：快速查找目标分类
   - 状态显示：直观查看映射状态（已映射/已删除/未映射）
   - 删除标记：将不需要的分类标记为删除，导入时自动过滤
3. **账户映射** - 配置原账户到目标账户的映射关系
4. **数据导出** - 导出为貔貅记账可导入的CSV格式

## 使用步骤

### 1. 启动程序

**macOS/Linux:**
```bash
./run_converter.sh
```

**Windows 或直接运行:**
```bash
python3 pixiu_converter.py
```

### 2. 准备依赖

确保已安装必要的Python包：

```bash
pip3 install pandas
```

### 3. 使用流程

#### 第一步：文件选择
1. 点击"浏览"按钮选择您的 `Result_2.csv` 文件
2. 确认文件信息显示正确
3. 确保貔貅模板文件存在于同一目录下

#### 第二步：配置分类映射
1. 在左侧列表选择源分类（从原文件中提取）
   - 状态列显示：已映射（绿色）/已删除（红色）/未映射（灰色）
2. 在右侧选择对应的目标分类（貔貅记账标准分类）
   - 可使用搜索框快速查找分类（实时过滤）
3. 点击"设置映射"按钮建立映射关系
4. 点击"标记为删除"可标记不需要导入的分类
   - 标记为删除的分类在导入时会被自动过滤
5. 点击"自动匹配"可以自动匹配相同名称的分类
6. 映射完成后会自动保存配置

#### 第三步：配置账户映射
1. 在左侧列表选择源账户
2. 在右侧输入或快速选择目标账户名称
3. 点击"设置映射"按钮建立映射关系
4. 映射完成后会自动保存配置

#### 第四步：导出数据
1. 点击"预览转换结果"查看转换后的数据
2. 确认无误后点击"导出CSV"
3. 选择保存位置和文件名
4. 导出完成后可将文件导入到貔貅记账中

## 配置文件

工具会自动生成以下配置文件：

- `category_mapping.json` - 分类映射配置
- `account_mapping.json` - 账户映射配置

这些配置会在使用过程中自动保存，下次打开时会自动加载。

## 文件格式说明

### 输入文件格式 (Result_2.csv)

必需列：
- `date` - 交易日期
- `income_type` - 收支类型（收入/支出）
- `category` - 分类
- `sub_category` - 子分类
- `account_name` - 账户名称
- `income_amount` - 金额

可选列：
- `remark` - 备注

### 输出文件格式 (貔貅记账格式)

输出文件包含以下列：
- `交易日期` - 交易日期
- `收支类型` - 收入/支出
- `收支项目` - 分类项目
- `金额` - 金额（正数）
- `账户名称` - 账户名称
- `标签` - 标签（默认为空）
- `备注` - 备注

## 常见问题

### Q: 如何批量导入分类映射？
A: 可以直接编辑 `category_mapping.json` 文件，格式为：
```json
{
  "收入|工资收入|工资": "工资",
  "支出|日常|正餐": "日常吃喝"
}
```

### Q: 如何将不需要的分类标记为删除？
A:
1. 在源分类列表中选择该分类
2. 点击"标记为删除"按钮
3. 状态会显示为"已删除"（红色）
4. 导入时会自动过滤这些记录

### Q: 导出的数据无法导入貔貅记账怎么办？
A: 请确保：
1. 文件编码为 UTF-8
2. 列名和顺序与貔貅模板一致
3. 日期格式正确（YYYY-MM-DD）
4. 金额为正数

### Q: 如何处理未映射的分类？
A: 在导出选项中勾选"导出未映射的记录"，程序会使用原分类名称作为目标分类。

### Q: 搜索框如何使用？
A: 在搜索框中输入关键词，目标分类列表会实时过滤显示匹配的分类。切换收支类型时会自动清空搜索框。

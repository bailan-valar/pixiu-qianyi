# 貔貅记账转换工具 - 快速开始

## 一键启动

### macOS用户
双击 `start_app.command` 文件即可启动

### 其他系统
```bash
python3 pixiu_converter.py
```

## 四步完成数据转换

### 1️⃣ 文件选择
- 点击"浏览"按钮
- 选择您的 `Result_2.csv` 文件
- 等待文件加载完成

### 2️⃣ 分类映射
- 左侧：选择原文件的分类（显示映射状态：已映射/已删除/未映射）
- 右侧：选择对应的貔貅记账分类
- 可使用搜索框快速查找目标分类
- 点击"设置映射"建立映射关系
- 点击"标记为删除"可忽略该分类（导入时自动过滤）
- 建议先点击"自动匹配"进行快速匹配

### 3️⃣ 账户映射
- 左侧：选择原文件中的账户
- 右侧：输入或快速选择目标账户
- 点击"设置映射"

### 4️⃣ 导出数据
- 点击"预览转换结果"查看数据
- 确认无误后点击"导出CSV"
- 保存文件

## 提示

- 所有映射配置会自动保存，下次使用时自动加载
- 可以多次点击"自动匹配"来完善映射关系
- 导出时建议勾选"导出未映射的记录"选项

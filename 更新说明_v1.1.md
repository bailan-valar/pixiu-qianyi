# 貔貅记账转换工具 - 功能更新说明

## 版本 v1.1 - 2026-04-22

### 🎉 新增功能

#### 1. 目标分类搜索功能
- **位置**：分类映射标签页右侧
- **功能**：实时搜索过滤目标分类
- **使用方法**：
  - 在搜索框中输入关键词
  - 列表会自动显示匹配的分类
  - 切换收支类型时自动清空搜索框

#### 2. 源分类状态显示
- **位置**：分类映射标签页左侧
- **功能**：直观显示每个源分类的映射状态
- **状态类型**：
  - 🟢 **已映射**（绿色）：已设置目标分类
  - 🔴 **已删除**（红色）：标记为删除，导入时自动过滤
  - ⚪ **未映射**（灰色）：尚未设置映射

#### 3. 删除标记功能
- **位置**：分类映射标签页右侧
- **功能**：将不需要导入的分类标记为删除
- **使用方法**：
  1. 在左侧源分类列表中选择要删除的分类
  2. 点击"标记为删除"按钮
  3. 状态变为"已删除"（红色）
  4. 导入时这些记录会被自动过滤
- **应用场景**：
  - 排除测试数据
  - 过滤不需要的账户类型
  - 清理无关的分类记录

### 🔧 改进功能

#### 分类映射优化
- 映射状态实时更新显示
- 删除映射时状态自动恢复为"未映射"
- 切换分类时自动更新映射信息显示

#### 导出功能增强
- 导出时显示过滤统计信息
- 自动过滤标记为删除的记录
- 控制台输出处理进度

### 📝 配置文件格式更新

#### category_mapping.json
```json
{
  "收入|工资收入|工资": "工资",
  "支出|日常|正餐": "日常吃喝",
  "支出|测试|测试数据": "__DELETE__"
}
```

**说明**：
- `__DELETE__` 为特殊标记，表示该分类在导入时会被过滤

### 🎯 使用技巧

#### 快速映射流程
1. 点击"自动匹配"完成初步映射
2. 使用搜索框快速查找目标分类
3. 对不需要的分类点击"标记为删除"
4. 检查状态列，确认所有分类都已处理

#### 批量处理建议
- 先处理主要的收入/支出分类
- 再处理特殊分类（如测试数据）
- 最后检查所有"未映射"状态的项目

### 🐛 已知问题修复
- 修复状态显示不更新问题
- 修复删除映射后状态不恢复问题
- 优化搜索性能

### 📊 数据处理流程

```
源数据
  ↓
检查映射状态
  ↓
├─ 已删除 → 过滤掉
├─ 已映射 → 使用映射的分类
└─ 未映射 → 根据选项决定是否导出
  ↓
生成目标数据
  ↓
导出CSV
```

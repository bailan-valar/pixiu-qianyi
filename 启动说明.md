# 🚀 貔貅记账转换工具 - 启动指南

## 一键启动

### macOS 用户
1. 双击 `start_app.command` 文件
2. 如果提示权限问题，在终端执行：
   ```bash
   chmod +x start_app.command
   ```

### Windows 用户
1. 确保已安装 Python
2. 双击运行 `start_app.bat` 或在命令行执行：
   ```cmd
   python pixiu_converter.py
   ```

### Linux 用户
1. 执行启动脚本：
   ```bash
   ./run_converter.sh
   ```
2. 或直接运行：
   ```bash
   python3 pixiu_converter.py
   ```

## 首次使用

### 1. 安装依赖
```bash
pip3 install -r requirements.txt
```

### 2. 准备文件
- 将您的 `Result_2.csv` 文件放在同一目录
- 确保 `貔貅记账#日常收支交易导入模版.csv` 文件存在
- 确保 `target_category.json` 文件存在

### 3. 启动程序
按照上述方法启动程序

### 4. 按照四步流程操作
1. 📁 **文件选择** - 选择源文件
2. 🏷️ **分类映射** - 配置分类映射关系
3. 💰 **账户映射** - 配置账户映射关系
4. 📤 **导出数据** - 预览并导出结果

## 配置文件

程序会自动生成以下配置文件：
- `category_mapping.json` - 分类映射配置
- `account_mapping.json` - 账户映射配置

可以手动编辑这些文件，或通过程序界面进行配置。

## 常见问题

### Q: 程序无法启动？
A: 检查 Python 版本（需要 3.6+）和依赖包

### Q: 文件加载失败？
A: 确保 CSV 文件格式正确，包含必需的列

### Q: 导出后无法导入貔貅记账？
A: 检查日期格式、金额格式，确保与貔貅模板一致
